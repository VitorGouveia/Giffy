## Props

The `Featured` component has 8 props

- tags, an array of [tags](#tags-type)
- title, the title of the post
- type, the type of the post
- description, the description of the post
- createdAt, a [ISO string Date](#iso-date-javascript) of when the post was created
- readTime, an approximate of how much it would take to read the post
- tagsPath, tags page url
- variant, the size variant of the post
